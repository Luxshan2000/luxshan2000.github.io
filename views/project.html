    Email Client

    I have developed an academic module project called 'Email Client Console Application'. The application allows users to add new contacts with their details, including their birthdays, and send emails to them using the console. The application also automatically sends birthday wishes to the contacts. In addition, the application has a feature to check the history of sent emails. By developing this project, I was able to improve my understanding of object-oriented programming concepts and design patterns in Java. I used the JavaMail API to achieve the email functionalities in the application.
